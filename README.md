# BF IDE

A simple IDE for the programming language BF, made using [Dear ImGui](https://github.com/ocornut/imgui)

The code is divided into 2 main components:
- [App](App/README.md): the actual IDE, where you can edit, compile, emulate the code, see the values in
       the memory and more.
- [Compiler](Compiler/README.md): provides both a standalone program and an API for the compilation

The compiler actually supports a custom extension of the language, see the compiler README.
